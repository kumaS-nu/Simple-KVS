    Class InMemoryKVS<T>

    Key Value Store works on memory. This can not use for subclass of UnityEngine.Object . If you want to use for subclass of UnityEngine.Object , use InMemoryKVS4UnityObject<T>.

    メモリ上で動作するKey Value Store。UnityEngine.Objectを継承していると使えない。その場合はInMemoryKVS4UnityObject<T>を使用。
